# DigiGrows Up 🚀

A full website built for **The DigiGrows Up**, a performance-first digital 
marketing agency based in Raipur, Chhattisgarh. The site positions the brand 
as a data-driven growth partner for ambitious businesses across SEO, PPC, 
social media, and branding.

## 🔗 Live Site
[digi-grows-up.vercel.app](https://digi-grows-up.vercel.app/)

## 🛠️ Built With
- Next.js
- TypeScript
- Tailwind CSS
- Vercel (deployment)

## 📦 Features
- Multi-page website (Home, Services, Work, About, Contact)
- Services showcase — SEO, Performance Marketing, Social Media, Branding
- Case studies section with real metrics and results
- Client testimonials
- Stats section (ROAS, CTR, Revenue Generated)
- Newsletter subscription
- WhatsApp, Instagram & Email contact integration
- Privacy Policy & Terms pages
- Fully responsive design

## 🏢 Client
The DigiGrows Up — Raipur, Chhattisgarh, India
